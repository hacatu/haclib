\begin{DoxyCompactList}\small\item\em Sorts a vector. Uses quicksort, modified to fall back to selection sort if the size of the part being sorted is less than or equal to 7. Because it is written using only macros, there is obviously no recursion. Instead, the left half is iteratively quicksorted and the right half is popped onto a vector of start/end pairs to be processed when this iteration is finished. This is looped as long as the vector of pairs is not empty. \end{DoxyCompactList}\end{DoxyCompactItemize}

Sorts a vector. Uses quicksort, modified to fall back to selection sort if the size of the part being sorted is less than or equal to 7. Because it is written using only macros, there is obviously no recursion. Instead, the left half is iteratively quicksorted and the right half is popped onto a vector of start/end pairs to be processed when this iteration is finished. This is looped as long as the vector of pairs is not empty.
