Applies expr to a value and every element of a vector in order, updating the value and then returning it. Note: uses __auto_type, a nonstandard C extension, to determine the type of the accumulator/return value.

Parameters
base_tthe base type.
veca pointer to the vector to fold.
initthe initial value of the accumulator. The type of this parameter will be the return type, and expr must process a value of this type and base_t.
expran expression containing e (the current element) and l (the last accumulator value, starts as init).
Returns
the final accumulator value.

Create a vector that is an arithmetic sequence of additive values (ie numeric type values). Starts at i1, adds step until i2 is reached, and pushes each value to the end of the vector.

Parameters
base_tthe base type. Must be valid to use with +.
i1the starting value (inclusive).
i2the ending value (exclusive).
stepthe step size.
Returns
a vector that is an arithmetic sequence with a0 = i1, d = step, and stopping just before i2.
