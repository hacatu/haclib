Applies expr to a value and every element of a sequence in reverse order, updating the value and then returning it. Note: uses __auto_type, a nonstandard C extension, to determine the type of the accumulator/return value.

Parameters
base_tthe base type.
seqa pointer to the sequence to fold.
initthe initial value of the accumulator. The type of this parameter will be the return type, and expr must process a value of this type and base_t.
expran expression containing e (the current element) and l (the last accumulator value, starts as init).
Returns
the final accumulator value.
